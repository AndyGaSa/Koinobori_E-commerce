PASAPALABRA

¡Hola! Bienvenido/a a Pasapalabra, el juego en el que tienes que completar el rosco para ganar. Por cada letra se te presentará una definición y tendrás que acertar de qué palabra se trata. Tras escribir tu respuesta, pulsa el botón de enviar respuesta para comprobar si has acertado (o también puedes pulasr la tecla 'Enter'). Si no sabes la palabra y deseas dejarla para más tarde, pulsa el botón de Pasapalabra.

Recuerda que tienes un tiempo límite para resolver el rosco al completo. Para empezar, escribe tu nombre y la cantidad de segundos con los que quieres empezar la partida. Cuando estés listo, presiona el botón de jugar. ¡Ánimo!
